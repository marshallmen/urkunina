# Sistema de Información para la Gestión de la tienda Urkunina

Este proyecto tiene como objetivo desarrollar un sistema de información que soporte la gestión de una tienda o emprendimiento local, mejorando los procesos de inventario y ventas.

## Módulos

1. **Módulo de Inventario**:
   - Registro, listado y modificación de productos.
   - Visualización del catálogo de productos.

2. **Módulo de Ventas**:
   - Gestión del carrito de compras.
   - Confirmación de compras y actualización de stock.
   - Listado de ventas realizadas.

## Estructura del Proyecto

- `src/` - Código fuente de la aplicación.
  - `src/inventory/` - Código relacionado con el módulo de inventario.
  - `src/sales/` - Código relacionado con el módulo de ventas.
- `docs/` - Documentación del proyecto.
- `tests/` - Pruebas del sistema.
